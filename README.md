# Employee Attendance System – MS Project 2016

## Project Overview

This repository contains the complete project management schedule for an **Employee Attendance System** developed using **Microsoft Project 2016**. The project demonstrates the planning, scheduling, resource allocation and execution phases involved in developing an attendance management solution.

The Employee Attendance System is designed to automate employee attendance tracking using technologies such as biometric devices, RFID cards, or facial recognition systems.

## Objectives

* Automate employee attendance recording.
* Reduce manual attendance errors.
* Improve attendance monitoring and reporting.
* Integrate biometric or RFID devices.
* Generate attendance reports for management.
